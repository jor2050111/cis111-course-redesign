# Module 10: Cybersecurity and Risk Assessment

Data breaches expose millions of records every year, costing organizations billions of dollars and eroding public trust. Behind every breach is a chain of decisions, some technical, some organizational, and many ethical. Understanding how to assess risk, formulate security policies, and respond to incidents is essential for any IT professional.

This module covers the ethical dimensions of cybersecurity, including the evaluation of security breaches, risk assessment frameworks, security policy development, and incident response planning. You will work with multiple CLOs as you develop the ability to evaluate, formulate, and plan for the security challenges that IT professionals face daily.

## Module Overview 🧭

* **Estimated time:** 5-6 hours (reading + activities + discussion)
* **Prerequisites:** Modules 1-9
* **Deliverables:** Discussion post, Quick Checks, Skills Lab analysis
* **CLOs addressed:** CLO XV, CLO XVI, CLO XVIII

## Learning Objectives 🎯

By the end of this module, you will be able to:

* **10.1-LO1 (Evaluate):** Evaluate IT security breaches by identifying root causes, impacts, and lessons learned (CLO XV)
* **10.2-LO2 (Create):** Formulate a security policy that addresses identified risks and aligns with organizational goals (CLO XVI)
* **10.3-LO3 (Apply):** Develop a prevention, detection, and response plan for a given IT security threat scenario (CLO XVIII)
